## 下载

### Docker

```
docker pull apachecn0/apachecn-angular-zh
docker run -tid -p <port>:80 apachecn0/apachecn-angular-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-angular-zh
apachecn-angular-zh <port>
# 访问 http://localhost:{port} 查看文档
```
