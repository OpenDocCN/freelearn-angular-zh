## 下载

### Docker

```
docker pull apachecn0/freelearn-angular-zh
docker run -tid -p <port>:80 apachecn0/freelearn-angular-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-angular-zh
freelearn-angular-zh <port>
# 访问 http://localhost:{port} 查看文档
```
